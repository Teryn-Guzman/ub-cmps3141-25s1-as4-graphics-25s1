Good Contrast(A)

This flyer follows the principle of contrast by using bright neon text and graphics against a dark background, making key information highly visible and attention-grabbing.

Violates Contrast(A)

Explanation:

This design violates the principle of contrast because many text elements are placed over bright, busy backgrounds using colors that are too similar in brightness and saturation, making the text difficult to read.

How to improve:

Contrast could be improved by using darker text on light backgrounds , adding solid color overlays behind text, and limiting the color palette for typography so important messages stand out clearly.

Violates Proximity(B)

Explanation:

This poster violates proximity because related information such as date, time, location, and activities are scattered across the layout instead of being grouped together.

How to improve:

Grouping related details into clear sections (event info, activities, location) and increasing spacing between unrelated elements would make the poster easier to scan and understand
